# 中華穿越鏡 (Hanfu AI Try-On) 🏮

「中華穿越鏡」是一款基於常見 AI 技術的動態歷史換裝應用。用戶只需拍攝一張照片，即可瞬間「穿越」時空，穿上唐、宋、明、清等不同朝代的傳統服飾，親身感受中華文化的厚重與美學。

---

## 🏫 適合場景：校園「中華文化日」活動

本項目專為學校教育、文化博覽、社區活動設計，具備以下教育價值：
- **沉浸式體驗**：通過 AI 換裝，讓學生從「觀察者」變為「參與者」，激發對中華傳統服裝的興趣。
- **寓教於樂**：内置「服飾百科」，在換裝的同時學習各朝代的歷史背景、服飾特點與文化內涵。
- **互動互動性強**：即拍即生成，一鍵保存/分享成果，是校園活動、展位攤位的「吸睛」利器。

---

## ✨ 核心亮點

### 1. 深度面部保護算法 🛡️
採用先進的圖像編輯模型指令，**強制保留用戶原始五官特徵**。換裝後臉型、神態與本人 100% 一致，確保「穿越」體驗真實自然，不變形、不換臉。

### 2. 多朝代精確適配 👗
- **大唐盛世**：齊胸襦裙、圓領袍。
- **風雅大宋**：清雅褙子、襴衫。
- **大明風華**：馬面裙、襖裙。
- **清宮舊夢**：旗裝、長袍馬褂。

### 3. 智能場景切換 🏞️
支持**「歷史背景」與「實景拍攝」**雙模式切換。既可以選擇 AI 生成的古風建築背景，也可以保留校園實景，僅更換服飾，滿足不同拍攝需求。

### 4. 極簡交互設計 📱
- **正方形取景框**：大尺寸「上下貼邊」引導框，幫助學生輕鬆對準構圖。
- **大按鈕優化**：針對移動設備優化的超大操作按鈕，適配各類手機與平板電腦。
- **百科查閱**：一鍵進入百科頁面，獲取最權威的服飾科普信息。

---

## 🛠️ 技術架構

- **前端**：React 18 + TypeScript + Vite + Tailwind CSS
- **後端**：Node.js + Express (部署於 Vercel Serverless)
- **AI 引擎**：阿里雲 DashScope (通義萬相 qwen-image-edit-max)
- **圖標庫**：Lucide React

---

## 🚀 快速開始

### 環境變量配置 (Environment Variables)

本項目支持多種環境變量以自定義功能和 UI。請在項目根目錄創建 `.env` 文件（或在 Vercel 後台配置）：

#### 核心配置 (Required)
| 變量名 | 說明 | 示例值 |
| :--- | :--- | :--- |
| `QWEN_API_KEY` | **[必須]** 阿里雲 DashScope API 密鑰 | `sk-xxxxxxxx` |

#### UI 與活動定制 (Optional)
| 變量名 | 說明 | 示例值 |
| :--- | :--- | :--- |
| `VITE_EVENT_NAME` | 活動名稱 (將顯示在標題和瀏覽器標籤頁) | `華夏衣冠文化節` |
| `VITE_SCHOOL_NAME` | 學校/機構名稱 (顯示在右上角) | `香港中文大學` |
| `VITE_SCHOOL_LOGO_URL` | 學校/機構 Logo 圖片鏈接 | `https://example.com/logo.png` |
| `VITE_EVENT_DATE` | 活動日期 (顯示在右上角) | `2024.10.01` |
| `ALLOWED_ORIGINS` | [後端] 允許的 CORS 來源 (多個用逗號分隔) | `https://your-frontend.vercel.app` |

### 安裝與運行
```bash
# 安裝依賴
npm install

# 啟動開發服務器 (同時啟動前後端)
npm run dev

# 構建生產環境
npm run build
```
